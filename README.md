Wikipedia Web Application

A responsive web application that enables users to search and explore Wikipedia articles in real time using the official Wikipedia API.

Overview

This project provides a clean and intuitive interface for searching Wikipedia content. Users can enter a query and instantly receive relevant article results, including summaries and direct links to the full pages on Wikipedia.

The application focuses on simplicity, performance, and user experience while demonstrating effective API integration and dynamic content rendering.

Features

Real-time search functionality

Dynamic data fetching from the Wikipedia API

Responsive design for desktop and mobile devices

Direct links to full Wikipedia articles

Basic error handling for empty or invalid searches

Clean and user-friendly interface

How It Works

The user enters a search term into the input field.

The application sends a request to the Wikipedia API.

Results are returned in JSON format.

The interface dynamically renders article titles, summaries, and links.

Users can click on any result to view the full article on Wikipedia.

Installation and Setup

To run this project locally:

git clone https://github.com/JayM5121/Wikipidedia-Web-Application.git
cd Wikipidedia-Web-Application


Open the index.html file in your preferred web browser.

No additional dependencies or installations are required.
